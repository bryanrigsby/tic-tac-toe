Tic-Tac-Toe

Welcome to Tic-Tac-Toe, an addictive game that can be traced back to ancient Egypt. In 1952, Tic-Tac-Toe became one of the first known computer games thanks to Alexander S. Douglas at the University of Cambridge. The version you see before you was made using React, a JavaScript library for building user interfaces, and Bootstrap, an open source toolkit for developing with HTML, CSS, and JS. The code came from the tutorial at reactjs.org, modified by me. The historical information about Tic-Tac-Toe came from wikipeidia.
